### 简介

    项目采用MVVM架构开发，Dotnet版本为8.0。

#### 主要使用到的第三方库如下
    * CommunityToolkit.Mvvm
    * NLog
    * RestSharp
    * Semi.Avalonia 
    * Ursa.Avalonia
